# Logo Anteas Lucca

## Asset Disponibili

### Logo Principale
- **logo.svg** - Logo vettoriale SVG (uso principale)
- **link.png** - Logo in formato PNG
- **logo.ai** - File sorgente Adobe Illustrator

### Favicon & Icone
Nella cartella `public/` (root):
- `favicon-16x16.png` - Favicon 16x16
- `favicon-32x32.png` - Favicon 32x32
- `apple-touch-icon.png` - Icona Apple (180x180)
- `android-chrome-192x192.png` - Icona Android
- `site.webmanifest` - Manifest PWA

### Altri Format
- `SVG/Asset 1.svg` - Versione alternativa SVG
- `link.ai` - File Adobe Illustrator alternativo

## Utilizzo

### In HTML
```html
<!-- Logo principale -->
<img src="public/logo/logo.svg" alt="Anteas Lucca" class="w-12 h-12">

<!-- Favicon -->
<link rel="icon" type="image/png" sizes="32x32" href="public/favicon-32x32.png">
```

### Colori Logo
- Verde: `#06593d`
- Rosso: `#d62a22`
- Bianco: `#ffffff`
